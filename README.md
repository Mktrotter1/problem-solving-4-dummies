# Problem Solving 4 Dummies

A cookbook for solving problems from zero.

You're standing in front of something. You have no information. Here's what to do.

## The Sequence

Every problem-solving framework ever invented is a variation on this sequence. The sequence is the thing. The frameworks are tools you reach for when a particular step is hard.

| Step | Name | Question You're Answering |
|------|------|---------------------------|
| [01](01-notice/) | **Notice** | Is there a problem? |
| [02](02-observe/) | **Observe** | What's actually happening? |
| [03](03-define/) | **Define** | What exactly is the problem? |
| [04](04-understand/) | **Understand** | Why is it happening? |
| [05](05-define-done/) | **Define Done** | What does solved look like? |
| [06](06-generate-options/) | **Generate Options** | What could I do about it? |
| [07](07-choose/) | **Choose** | Which option do I pick? |
| [08](08-act/) | **Act** | Do the thing. |
| [09](09-check/) | **Check** | Did it work? |
| [10](10-reflect/) | **Reflect** | What did I learn? |

## How to Use This

**When you're stuck**, find which step you're actually on. You're almost certainly stuck because you skipped one of the earlier steps. Go back.

**When you're overwhelmed**, the answer is always step 3. You haven't defined the problem clearly enough yet.

**When nothing is working**, the answer is usually step 4. You're solving a symptom, not the problem.

## Where the Frameworks Live

The classic frameworks aren't separate things to learn. They're tools that help with specific steps:

| Framework | Helps With |
|-----------|------------|
| Polya's Four Steps | Steps 3, 6, 8, 10 |
| Scientific Method | Steps 2, 3, 6, 8, 9 |
| OODA Loop | Steps 2, 4, 7, 8 (looped fast) |
| IDEAL Model | Steps 3, 4, 6, 8, 10 |
| Design Thinking | Steps 2, 3, 6, 8, 9 |
| 5 Whys | Step 4 |
| First Principles | Step 4 |
| Feynman Technique | Steps 3, 4 |
| Rubber Duck | Steps 3, 4 |
| Inversion | Steps 5, 6 |
| Occam's Razor | Step 7 |
| Cynefin Framework | Calibrates how much effort steps 4-6 need |
| Bloom's Taxonomy | Tells you which cognitive level you're operating at |
| Six Thinking Hats | Step 6 (forces multiple perspectives) |
| TRIZ | Step 6 (for inventive/novel problems) |
| Fishbone / Ishikawa | Step 4 (maps causes by category) |
| PDCA / Deming Cycle | Steps 8, 9, 10 (looped) |
