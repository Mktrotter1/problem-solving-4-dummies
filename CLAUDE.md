# Problem Solving 4 Dummies

A cookbook/reference repo for problem-solving and critical thinking fundamentals.

## Structure

10 sequential steps, one directory each (`01-notice/` through `10-reflect/`). Each step has a `README.md` explaining:
- What the step is and why it matters
- How to do it practically
- Which classic frameworks/techniques help with this step
- Common mistakes

## Principles

- **Sequence matters.** The 10 steps follow the natural cognitive order from zero knowledge to solved problem. Don't reorganize them.
- **Frameworks serve steps.** Classic frameworks (Polya, OODA, 5 Whys, etc.) are nested under the steps they help with, not presented as standalone things to memorize.
- **Practical over academic.** This is a cookbook, not a textbook. Write for someone who is stuck right now.
- **Assume zero prior knowledge.** The reader has a problem and no information. Start there.

## Writing Style

- Direct, concise, conversational
- Use examples over abstractions
- "Do this" over "one might consider"
- Keep each step's README self-contained but cross-reference freely
